# eTeacherProject Frontend

This is the **frontend application** for e‑learning platform. 
It is built with **React** and integrates with the backend API to manage **Students, Courses, and Enrolments**.  
The frontend provides a simple and intuitive interface for users to interact with the system.

---

## Key Features

- **Students Management**
  - View all students
  - Create and update students
  - Display student details (name, phone, address)

- **Courses Management**
  - View all courses
  - Create and update courses
  - Display course details (title, description, lecturer)

- **Enrolments Management**
  - View all enrolments
  - Create and update enrolments
  - Assign students to enrolments
  - Generate enrolment reports with student lists

- **UI/UX**
  - Built with **Material‑UI (MUI)** for a modern, responsive design
  - Dialogs, tables, and forms for smooth user interaction
